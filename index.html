Installation — GunzOTC legacy

  1. Use an old backup where hotkeySets still contains your bars (e.g. .pre_import_*, .bak_*). The current character file often has hotkeySets: [] — that cannot be migrated here.
  2. Close GunzOTC. Open %APPDATA%\Gunzodus\GunzOTC\settings\
  3. Copy all clientoptions_preset_*.json from the ZIP.
  4. Replace your character clientoptions_*.json with the new one (points to the active preset, bars live in preset files).
  5. Merge clientoptions_profiles.json if provided, or pick the preset under Options → Controls → Keybinds.
  6. Launch the game and confirm icons + hotkeys on the action bar.

Installation — OTClientV8

  1. Find actionbar_v2.json in %APPDATA%\OTClientV8\otclientv8\settings\profile_N\ (N = profile number in OTCv8).
  2. Upload it here. Set preset name = your character name for easiest setup.
  3. Close GunzOTC. Copy files from the ZIP to %APPDATA%\Gunzodus\GunzOTC\settings\
  4. Launch GunzOTC → log in → Options → Controls → Keybinds → select your preset (or it auto-selects if the name matches your character).
